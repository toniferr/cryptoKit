# cryptoKit

## English
Java Development in Swing technology for encryption / decryption using distinct algorithms:

Java version: 11

Algorithms implemented:

- Base64
- Hash -> MD5, SHA1
- Symmetric -> AES (ECB,CBC) and DES

External libraries: 
- bcprov-jdk15on-165
- jaxb-api-2.3.1
