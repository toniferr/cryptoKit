# cryptoKit

## English
Java Development in Swing technology for encryption / decryption using distinct algorithms:

Algorithms implemented:

- Base64
- Hash -> MD5, SHA1
- Symmetric -> AES (ECB,CBC) and DES

External library: bcprov-jdk15on-157

Algoritmos implementados:
